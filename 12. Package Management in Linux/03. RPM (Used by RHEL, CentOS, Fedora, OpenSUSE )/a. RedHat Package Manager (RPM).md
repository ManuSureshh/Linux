## How to find the RPM Packages
- Red Hat developed RPM packages can be found at
  - Red Hat Enterprise Linux CD ROMs
  - Red Hat Network
  - Red Hat errara page having list of packages
- It can be found on the internet.
  - http://rpmfind.net
  - http://www.redhat.com
  - http://rpm.pbone.net
